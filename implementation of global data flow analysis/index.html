📖 Syntax & Semantics

• Blocks: Labeled with a colon (e.g. B1:, Loop:).
• Definitions: Assignments like x = y + z will automatically be assigned an ID (e.g. d1, d2).
• Control Flow: Use goto BlockName or if cond goto BlockName. Blocks fall through automatically to the next block if no unconditional goto is present.
